跨平台技术

## 原生平台

移动平台主要有 Android 和 iOS 两个大平台：

- iOS（Google）
- Android（Apple）
- Windows Phone（微软，2019/12/10停止更新）



## 跨平台技术方案



- H5 + 原生开发（Android/iOS）：这种技术叫混合开发，微信小程序、Cordova、原生WebView直接嵌套H5
- JavaScript + 原生渲染：React Native、Weex
- 自绘制UI + 原生：Qt for mobile、Flutter
